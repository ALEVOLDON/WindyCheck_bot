# 🌬️ WindyCheck Telegram Bot

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![aiogram 3.x](https://img.shields.io/badge/aiogram-3.x-blueviolet.svg)](https://docs.aiogram.dev/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**WindyCheck Bot** — это асинхронный Telegram-бот на Python (`aiogram 3` + `aiohttp`), предназначенный для отслеживания ветра, порывов, прогнозов погоды, визуализации розы ветров и авто-уведомлений при сильном ветре.

Идеально подходит для виндсёрферов, кайтсёрферов, яхтсменов, парапланеристов и любителей активного отдыха.

---

## 🛠️ Функциональные возможности

- 🌬️ **Текущий ветер:** Скорость (м/с), порывы, направление (градусы + визуальная стрелка 🧭), температура.
- 📅 **Прогноз 24 часа:** Почасовой прогноз изменения параметров ветра на ближайшие сутки.
- 📊 **Графики и диаграммы:** Генерация PNG-картинок с графиками скоростей ветра и **розой ветров** (`matplotlib`).
- 🗺️ **Интерактивные карты:** Генерация ссылок на живую карту погоды [Windy.com](https://www.windy.com).
- 📍 **Избранное (Мои города):** Сохранение персонального списка спотов для каждого пользователя.
- 📊 **Сравнение спотов:** Сравнительный график трендов ветра сразу по нескольким городам.
- 🔔 **Авто-уведомления (Алерты):** Фоновый мониторинг каждые 30 минут (`asyncio`) и отправка push-оповещений при превышении порога скорости ветра.

## 🚀 Быстрый запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/your-username/WindyCheck_bot.git
cd WindyCheck_bot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
Скопируйте файл `.env.example` в `.env` и укажите ваши токены:
```bash
cp .env.example .env
```

Отредактируйте `.env`:
```env
BOT_TOKEN=your_telegram_bot_token_here
OPENWEATHER_API_KEY=your_openweather_api_key_here
```

### 4. Запуск бота
```bash
python bot.py
```

## 📋 Команды бота

| Команда | Описание |
| :--- | :--- |
| `/start` | Запуск бота и Главное меню |
| `/wind [город]` | Текущий ветер |
| `/forecast [город]` | Прогноз ветра на 24 часа + график |
| `/chart [город]` | График изменения скорости ветра |
| `/rose [город]` | Диаграмма «Роза ветров» |
| `/map [город]` | Ссылка на карту Windy.com |
| `/track [город]` | Добавить город в Избранное |
| `/untrack [город]` | Удалить город из Избранного |
| `/mywind` | Ветер по всем избранным городам |
| `/alert [скорость]` | Оповещение при ветре > X м/с |
| `/alert off` | Отключить оповещения |
